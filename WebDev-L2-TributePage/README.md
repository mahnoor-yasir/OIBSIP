# Tribute Page

A modern, responsive, and visually engaging Tribute Page developed as part of the **Oasis Infobyte Web Development & Designing Internship (Level 2 - Task 2)**. This project pays tribute to an inspiring personality through elegant design, engaging storytelling, and responsive web development techniques.

## Features

- Elegant Hero Section
- Responsive Navigation
- Biography Section
- Timeline of Major Achievements
- Inspirational Quote Section
- Image Gallery
- Attractive UI/UX
- Smooth Animations
- Responsive Design
- Cross-Browser Compatibility
- Semantic HTML5 Structure
- Clean CSS3 Styling

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (for optional interactions)

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html**.
4. The Tribute Page will open in your default web browser.

No installation or additional software is required.
