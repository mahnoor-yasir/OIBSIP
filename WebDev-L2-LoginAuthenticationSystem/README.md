# Login Authentication System

A secure, responsive, and user-friendly Login Authentication System developed as part of the **Oasis Infobyte Web Development & Designing Internship (Level 2 - Task 4)**. This project provides user registration, login authentication, session management, and a protected dashboard using modern web technologies.

## Features

- User Registration
- Secure Login Authentication
- Password Validation
- Username & Email Validation
- Duplicate User Detection
- Protected Dashboard
- Session Management
- Logout Functionality
- SHA-256 Password Hashing
- Local Storage Support
- Responsive Design
- Modern & Clean User Interface
- Form Validation
- Error Handling
- Cross-Browser Compatibility

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (ES6)
- Web Crypto API (SHA-256)
- LocalStorage API
- SessionStorage API

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html**.
4. Register a new account or log in with an existing account.

No installation or additional software is required.
