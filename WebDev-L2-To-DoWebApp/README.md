# To-Do Web App

A modern, responsive, and interactive To-Do Web Application developed as part of the **Oasis Infobyte Web Development & Designing Internship (Level 2 - Task 3)**. This application helps users efficiently manage their daily tasks with features such as task creation, editing, completion tracking, and local storage support.

## Features

- Add New Tasks
- Edit Existing Tasks
- Delete Tasks
- Mark Tasks as Completed
- Pending & Completed Task Lists
- Task Counter
- Timestamp Support
- Local Storage Persistence
- Empty State Messages
- Responsive Design
- Clean & Modern UI
- Smooth User Experience

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (ES6)
- LocalStorage API

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html**.
4. The application will open in your default web browser.

No installation or additional software is required.
